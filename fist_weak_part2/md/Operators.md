# Арифметические операторы в Java

**Арифметические операторы** используются для выполнения математических операций над числовыми типами данных.

## Основные арифметические операторы

| Оператор | Название | Пример | Результат |
|--------|---------|--------|----------|
| `+` | Сложение | `5 + 3` | `8` |
| `-` | Вычитание | `5 - 3` | `2` |
| `*` | Умножение | `5 * 3` | `15` |
| `/` | Деление | `5 / 2` | `2` (для int) |
| `%` | Остаток от деления | `5 % 2` | `1` |

Пример:
```java
int a = 10;
int b = 3;

System.out.println(a + b); // 13
System.out.println(a / b); // 3
System.out.println(a % b); // 1
```
# Конвертация между числовыми (numeric) типами
В Java существуют ***автоматические*** (неявные) и ***явные*** преобразования типов.

## Автоматическая (неявная) конвертация
Происходит, когда данные преобразуются из меньшего типа в больший:
```
byte → short → int → long → float → double
```

## Явная конвертация (casting)
Требуется, когда тип преобразуется из большего в меньший:
```java
double a = 9.8;
int b = (int) a; // b = 9
```

# ⚠️ Потеря данных при конвертации

`int → float`

`int` 32-битное целое

Диапазон: от −2 147 483 648 до 2 147 483 647

`float` — 32 бита

Из них только 24 бита используются для точного хранения целых чисел
(23 бита мантиссы + 1 скрытый бит)
```java
int a = 16_777_217; // больше 2^24
float f = a;

System.out.println(f); // 16777216.0
```
